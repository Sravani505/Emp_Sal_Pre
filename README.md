# 📌Project Title

Employee Salary Prediction in HRMS Using XGBoost with LLM and RAG

# 📖Project Description

This project predicts employee salaries using Machine Learning techniques and Large Language Models (LLMs). The system takes inputs such as experience, education, job role, and skills, and predicts salary using the XGBoost algorithm. It also generates explanations using RAG and LLM.

# 🎯Objectives

Predict employee salary using ML

Use LLM for understanding text inputs

Provide explanation for predictions

Build a simple interactive interface

# 🏗️System Architecture

Input: Employee details

Data Preprocessing

Feature Engineering

LLM Processing

XGBoost Model

Output: Predicted Salary + Explanation

# ⚙️Technologies Used

XGBoost, LLM
 
HTML, CSS
