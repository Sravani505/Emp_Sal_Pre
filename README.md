# Emp_Sal_Pre

📌 Project Title

Employee Salary Prediction Using Machine Learning and LLMs

📖 Project Description

This project aims to predict employee salaries using machine learning techniques and Large Language Models (LLMs). The system takes inputs such as experience, education, job role, and skills, and predicts salary using the XGBoost algorithm. Additionally, LLM and RAG are used to generate explanations for the predicted salary.

🎯 Objectives
Predict employee salary accurately using ML
Use LLM to analyze textual data (skills, job roles)
Provide explanation for predictions
Build an interactive system for users
🏗️ System Architecture
Input: Employee details
Preprocessing & Feature Engineering
LLM (for text understanding)
XGBoost Model
Output: Predicted Salary + Explanation
